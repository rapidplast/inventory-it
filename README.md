## Quick Installation

    git clone 

    cd SD_Project/

    code .
    
### Composer and npm

    composer update

    npm update

    npm run dev
    
    
### For Environment Variable Create
 
    cp .env.example .env
 
    
 ### For Migration table in database [Create database name as ```ims_new```] and Make Seeder
 
    php artisan migrate
    
    php artisan db:seed 
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
